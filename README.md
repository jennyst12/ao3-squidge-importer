# ao3-squidge-importer
Bookmarklets to import a fanfic from AO3 into SquidgeWorldArchive. Please only use on your own fics or those you are authorised to archive, and pay attention to the terms of both archives for what types of works are allowed - there are slight differences.

Created with help from Claude AI.

* Add both bookmarklet files to your bookmarks, or all four files for a multi-chapter fic.
* Make sure you're logged in to both AO3 and Squidge.
* Open a fic on AO3 in chapter view in your browser.
* Click first bookmarklet. It will copy the details of your fic and then navigate you to Squidge's new work form.
* Make sure the page has fully loaded.
* Click second bookmarklet. It will fill in all the fields it can. It will backdate the work to the original AO3 posting date. It won't include "Inspired by" or series, so add those yourself.
* Go through the tags, check and edit, select from the autocomplete where relevant, and hit enter at the end of each field to make them save.
* Post your work, or the first chapter of your work.
* For a multi-chapter work, then:
  * note down the ID of the work on Squidge
  * navigate to the second chapter on AO3
  * use the third bookmarklet to copy that chapter
  * use the fourth bookmarklet to paste
  * click post
  * naviate back to the next chapter on AO3
  * repeat third bookmarklet, fourth bookmarklet, post, until you've done all chapters
